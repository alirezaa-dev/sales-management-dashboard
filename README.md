# Sales Management Dashboard

A modern and responsive Sales Management Dashboard built with React, Vite, and Tailwind CSS. This project provides a clean and intuitive interface for managing customers, products, brands, categories, and orders.

---

## ✨ Features

- Responsive dashboard layout
- Customer management
- Product management
- Brand management
- Product category management
- Order management
- Sidebar navigation
- Modern and clean UI
- Reusable React components

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- JavaScript (ES6+)

---


## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/alirezaa-dev/sales-management-dashboard.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
